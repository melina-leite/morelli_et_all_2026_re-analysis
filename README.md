# Does sex really matter? Re-analysis of Morelli et al. (2026)

Re-analysis of the data from Morelli et al. (2026)[^1] for a commentary letter written by Melina de Souza Leite and Florian Hartig to the journal People and Nature.

[^1]: "Morelli, F., Benedetti, Y., Mikula, P., Blumstein, D. T., Díaz, M., Page, A., Tryjanowski, P., Nowak, M. K., Vincze, E., & Lövei, G. L. (2026). Sex matters: European urban birds flee approaching women sooner than approaching men. People and Nature, 8(2), 316–326. https://doi.org/10.1002/pan3.70226"

The data used in the analysis were published on [Figshare: https://doi.org/10.6084/m9.figshare.28284848.v1](https://doi.org/10.6084/m9.figshare.28284848.v1)[^2].

[^2]: "Morelli, Federico; Benedetti, Yanina; Mikula, Peter; Díaz, Mario; Tryjanowski, Piotr (2025). Urban birds FID and observers' sex. figshare. Dataset. https://doi.org/10.6084/m9.figshare.28284848.v1"

The phylogenetic bird trees data were shared by the corresponding author, Federico Morelli, via email on June 15, 2026. The same species tree could be downloaded from [birdtree.org](https://birdtree.org/) using the paper’s instructions.

You can see the analysis, results, and figures by opening (downloading) the HTML file `morelli_et_al_2026_re-analysis.htlm` or reproducing it by running the Quarto file `morelli_et_al_2026_re-analysis.qmd`.
